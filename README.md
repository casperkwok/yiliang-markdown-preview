### 为什么选择 Markdown 报告预览助手？

还在为表格中过长的报告内容而烦恼吗？需要反复拖动、切换窗口才能阅读全文？

这款 Markdown 报告预览工具将彻底解决您的痛点。它专为学生、研究人员、HR 及猎头等学术与职场人士设计，让您告别繁琐操作，专注于内容本身，显著提升工作效率。

### 核心功能

- **即时预览**：点击单元格，立即预览 Markdown 长文报告
- **一键导出**：支持导出 PDF 格式文件
- **简单操作**：无需复杂设置，即开即用
- **场景丰富**：适用于人才评估报告、学术实验报告等各类长篇文档
